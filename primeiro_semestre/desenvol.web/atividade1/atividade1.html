Lista de animais para doação

  1. Cachorros

    Golden retrivier
    Animal dócil e de porte médio.
    Alimentação: ração para animais de porte médio (quantidade 160g a 320g)

  2. Gatos

    Vira lata
    Animal perigoso, só sabe brincar com os dentes e unhas.
    Alimentação: ração para animais de porte pequeno (quantidade até 160g)

  3. Focas

    Não é permitido doar animais silvestres!!!
